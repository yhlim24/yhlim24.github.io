Lecture Overview

Media Interface course provides overall understanding of digital media and the foundation technologies that enable interactive media creation. Students will learn basic programming fundamentals and apply the technologies to develop interactive media prototyping. In this course, various prototyping tools such as Arduino, Little Bits or Processing will be used to create the interactive prototypes. The course will also cover HCI methods such as interaction design, prototyping, and usability testing for concept validation.

After taking this course, students will be able to develop their own sensors and computing environment to collect, analyze, and process data for their own journalism project. Also, students will be able to set and execute a sensor driven journalism research. Students are allowed to set up an innovative research project, able to conduct an experiment on their own, and knows how to make good data visualization upon the findings from the research.

Objectives

  • Understand the current digital media environment and technologies
  • Learn and apply Human-Computer Interaction methods to real-world problems
  • Apply the theoretical learnings to the actual concept building and prototyping
  • Master basic programming skills and handle development environment
